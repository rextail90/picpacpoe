# PyTacToe+ (CS 122)

A simple Tic-Tac-Toe with GUI (Tkinter) and AI (minimax with alpha-beta–style pruning).

## Requirements

- Python **3.10+** (you’re using 3.13, which is fine)
- Tkinter

## How to Run the Game

From the project root (the folder that contains the mainpackage directory):
```bash
python -m mainpackage.main
